# Royal Dutch Shell


To compile:

    $make

To run:

    $./royaldutch

features:

`pipe`, example:

    ls | sort | wc

`IO Redirection`, example:

    ls > file_list.txt
    cat < file_list.txt

`cd`, example:

    cd
    cd ~/
    cd ..

background, example:

    ping -c 10 example.com &

    fg
    ^z
    bg
